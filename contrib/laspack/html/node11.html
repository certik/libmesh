Program matropt

 

This program matropt could be used to check the efficiency of several implementations of the matrix by vector product: 

  <Vector 2> = <Matrix> <Vector 1>.
The results may be shown as follows: 
Results:    (for dimension 100000, cycles 100)
--------

 implementation    CPU time
-----------------------------------------
    LASPack       1.967e-01 s = 100.0 %
       1          2.836e-01 s = 144.2 %
       2          2.326e-01 s = 118.3 %
       3          1.928e-01 s =  98.0 %
       4          2.005e-01 s = 101.9 %
-----------------------------------------

For details of the implementations look at the file laspack/examples/matropt/testproc.c. The current LASPack version corresponds to the implementation 3. This is, in comparison with LASPack , simplified and can therefore be a little faster.
